How to create a theme

When creating a new theme on Pyxlpress, all you need to do is add a theme name and some general information about the theme. Once you create the theme, Pyxlpress will automatically generate the files necessary to activate the theme. You can modify these files as well as add more templates once the initial theme is created.

After the theme is created, you can edit it, add more pages and templates, modify the style as well as the JavaScript. If you wish, you can also download the theme, delete it, or re-upload a past backup.
